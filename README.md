# Zone XIV Business Directory Website

A premium, fully responsive, and feature-rich Business Directory Website built for Zone XIV.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/vdya1/zone-xiv-directory)

## 🌟 Key Features

- **Responsive Design**: Flawless visual grids optimized for Desktop, Tablet, and Mobile layouts.
- **Dynamic Search & Filtering**: Multi-condition searches by Business Name, Owner Name, Category, Chapter, and City.
- **Owner Profile CTAs**: Immediate direct communication links for WhatsApp chat, voice calls, emails, and external web addresses.
- **Admin Dashboard Panel**:
  - Secure Administrator login.
  - Approve pending registrations before they are published.
  - Full CRUD control (Add/Edit/Delete listings).
  - Manage Categories and Chapters taxonomy.
  - **Export to Excel**: Download active listings as a spreadsheet.
  - **JSON Backup & Restore**: Full database persistence control.
- **Theme Customizer**: Premium glassmorphic Dark Theme and slate Light Theme toggle switch.
- **Centralized Database**: Configured Node.js & Express server database that serves all users in real-time.

## 💻 Local Development

To run this project locally on your machine:

1. Clone this repository or open the project folder.
2. Install server dependencies:
   ```bash
   npm install
   ```
3. Start the Express server:
   ```bash
   npm start
   ```
4. Open your browser and navigate to:
   **[http://localhost:8080](http://localhost:8080)**
